How to Play

Objective: Move all the disks from the source tower to the destination tower while following the rules below.

Rules:

  • Only one disk can be moved at a time.
  • You can only place a disk on top of a larger disk or on an empty tower.
  • There are three towers: the source, the destination, and an auxiliary tower.
